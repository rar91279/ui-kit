# Typography

This page documents the font stacks used in the design system.

## Font Families

- **Sans-serif (Base)**: `system-ui, -apple-system, Segoe UI, Roboto, Helvetica, Arial, sans-serif, "Apple Color Emoji", "Segoe UI Emoji"`
- **Serif**: `Georgia, Cambria, "Times New Roman", Times, serif`
- **Monospace**: `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", monospace`

## Font Weights

- **Normal**: 400
- **Medium**: 500
- **Bold**: 700
