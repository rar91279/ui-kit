# Color Palette

This page demonstrates the dynamic color palette generated from a base color.
- **Theme**: Toggle between predefined themes like **Fanfare** or **Lava Falls**.
- **Color Scheme**: Manually switch between **Light** and **Dark** modes (or follow **System** preferences).
