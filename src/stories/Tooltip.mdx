# Tooltip

Tooltips are small overlays that provide extra information when hovering over an element.

## Usage

To add a tooltip to any element, use the `data-tooltip` attribute.

```html
<span data-tooltip="This is the tooltip text">Hover over me</span>
```

The tooltip is implemented entirely in CSS using the modern **CSS Anchor Positioning API** via the `::after` pseudo-element on elements that have the `[data-tooltip]` attribute.

## Properties

The tooltip behavior is controlled by the following CSS properties in `scss/module/tooltip.scss`:

- `data-tooltip`: The text to be displayed in the tooltip.
- `anchor-name`: Defines the base element as an anchor.
- `position-anchor`: Associates the tooltip with the base element anchor.
- `position-try-options`: Defines the fallback positions (`--bottom`, `--right`, `--left`).
- `@position-try`: Defines the custom position overrides for fallback scenarios.
- `bottom: anchor(top)` & `left: anchor(center)`: Positions the tooltip relative to the anchor.
- `::after`: The pseudo-element that creates the tooltip bubble.
- `transition`: Handles the fade-in effect on hover.
