# Tooltip

Tooltips are small overlays that provide extra information when hovering over an element.

## Usage

To add a tooltip to any element, use the `data-tooltip` attribute.

```html
<span data-tooltip="This is the tooltip text">Hover over me</span>
```

The tooltip is implemented entirely in CSS using the modern **CSS Anchor Positioning API** via the `::after` pseudo-element on elements that have the `[data-tooltip]` attribute.

### Scoping and Custom Positioning

The `Scoping` example demonstrates how multiple elements can have tooltips on the same page, even when using the same anchor name, by utilizing the `anchor-scope` property. It also shows how the tooltip's position can be customized using the `data-tooltip-position` attribute.

<Canvas of={TooltipStories.Scoping} />

#### `anchor-scope`

The `anchor-scope` property is used to limit the visibility of an anchor name to a specific subtree. In the tooltip implementation, `anchor-scope: --tooltip-anhor;` is applied to each element with the `[data-tooltip]` attribute. This ensures that the tooltip associated with each element correctly targets its own anchor, even if multiple elements share the same `--tooltip-anhor` name.

#### `data-tooltip-position`

The `data-tooltip-position` attribute allows for manual control over where the tooltip appears relative to its anchor. Supported values include `top`, `bottom`, `left`, `right`, `top-left`, `top-right`, `bottom-left`, `bottom-right`, `left-top`, `left-bottom`, `right-top`, and `right-bottom`.

## Properties

The tooltip behavior is controlled by the following CSS properties in `scss/module/tooltip.scss`:

- `data-tooltip`: The text to be displayed in the tooltip.
- `data-tooltip-position`: (Optional) Specifies the preferred position for the tooltip.
- `anchor-scope`: Scopes the `--tooltip-anhor` name to the element, preventing conflicts with other tooltips.
- `anchor-name`: Defines the element as an anchor named `--tooltip-anhor` on hover.
- `position-anchor`: Associates the tooltip (the `::after` pseudo-element) with the `--tooltip-anhor`.
- `position-area`: Positions the tooltip relative to its anchor using values like `top right` (default) or the value from `data-tooltip-position`.
- `::after`: The pseudo-element that creates the tooltip bubble.
